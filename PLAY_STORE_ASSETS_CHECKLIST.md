# Play Store Readiness Checklist

This prototype now includes core systems expected in a casual puzzle game:

- Match-3 gameplay loop
- Lives and refill timer
- Soft currency: Stars
- Coin economy
- Daily reward
- Boosters
- Garden renovation tasks
- Local save
- Optional backend leaderboard
- Privacy policy page
- PWA manifest and offline service worker

Still needed before real Play Store release:

- Build a real Android app or Trusted Web Activity wrapper
- 512x512 PNG app icon
- 1024x500 feature graphic
- At least 2 phone screenshots
- Production privacy policy hosted on a public URL
- Google Play Data Safety answers
- Content rating questionnaire
- Closed testing release
- Google Play Billing integration before selling coins for money
- Real backend hosting for leaderboard/cloud save
