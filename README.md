# MyGarden Match-3

A browser-playable Match-3 game prototype with:

- 8x8 match-3 board
- Bomb, rainbow, and dynamite power-ups
- Moves, score, target score, win/lose loop
- Stars, coins, lives, refill timer
- Garden task spending
- Local save
- Optional Node.js backend for profile sync and leaderboard

## Play

Open `index.html`.

## Backend

Run locally:

```bash
node backend/server.js
```

Then open:

```text
http://localhost:3000
```
